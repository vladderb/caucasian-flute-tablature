# Kamyl Tablature Plugin for MuseScore

Плагин для автоматического добавления аппликатуры (табулатуры) для камыля (черкесской флейты) в партитуры MuseScore.

### Диапазон камыля

**Основные ноты (диатонические):**
- **Регистр 0**: A3, Bb3, C4, D4
- **Мёртвая зона**: E4, F4, G4 не играются
- **Регистр 1**: A4, Bb4, C5, D5
- **Регистр 2**: E5, F5, G5, A5
- **Регистр 3**: Bb5, C6, D6

**Хроматические ноты (с полузакрытыми отверстиями ◑):**
- **Регистр 0**: B3, C#4/Db4
- **Регистр 1**: B4, C#5/Db5
- **Регистр 2**: F#5, G#5/Ab5
- **Регистр 3**: B5, C#6/Db6
- **Регистр 4**: D#6

**Примечание об альтернативных аппликатурах:** Некоторые ноты (A5, D6) имеют альтернативные аппликатуры в разных регистрах с разными пальцевыми позициями, плагин показывает их табулатуру в нижнем регистре (для A5 - во втором, для D6 - в третьем).

**Примечание о диапазоне воспроизводимых нот:** Ноты, которых нет в диапазоне камыля, будут отображаться красным цветом в MuseScore и не получат табулатуру.

### Установка

Подробная инструкция по установке инструмента Kamyl и плагина табулатуры находится в файле [INSTALL.md](INSTALL.md)


## Использование
1. Откройте партитуру в MuseScore
2. Выделите такты, к которым хотите добавить табулатуру (или не выделяйте ничего для обработки всей партитуры)
3. Перейдите в меню: Plugins → Kamyl Tablature
4. Табулатура будет автоматически добавлена под нотами

## Отмена изменений

Если результат вас не устраивает, используйте: Edit → Undo (Ctrl+Z) или примените Plugins → Remove Kamyl Tablature для удаления табов

## Особенности

- Табулатура размещается под нотным станом
- Табулатура добавляется для всех нот, включая связанные (tie)
- При повторном запуске плагина старые табы автоматически удаляются и заменяются новыми
- Некоторые ноты (A5, D6) имеют альтернативные аппликатуры в разных регистрах - плагин показывает только аппликатуру в нижнем регистре
- Плагин работает с любым инструментом - не обязательно менять инструмент на Kamyl
- Отдельный плагин для удаления всей табулатуры из партитуры

## Использование

### Добавление табулатуры

1. Откройте партитуру в MuseScore
2. Выделите такты, к которым хотите добавить табулатуру (или не выделяйте ничего для обработки всей партитуры)
3. Перейдите в меню: Plugins → Kamyl Tablature
4. Табулатура будет автоматически добавлена под нотами

### Удаление табулатуры

1. Откройте партитуру с табулатурой
2. Выделите такты, из которых хотите удалить табулатуру (или не выделяйте ничего для удаления из всей партитуры)
3. Перейдите в меню: Plugins → Remove Kamyl Tablature
4. Вся табулатура (кружки, регистры и названия нот) будет удалена

**Совет:** Используйте Edit → Undo (Ctrl+Z) для отмены изменений

## Известные ограничения

- Камыль играет только определенные ноты. Ноты вне диапазона или в "мёртвых зонах" будут отображаться с табулатурой "???" (три знака вопроса вместо кружков).
- Плагин работает только с инструментом Kamyl (kamyl или wind.flutes.kamyl)
- Требуется MuseScore версии 3.0 или выше

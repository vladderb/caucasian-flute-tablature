# Kamyl Tablature Plugin for MuseScore

Плагин для автоматического добавления аппликатуры (табулатуры) для камыля (черкесской флейты) в партитуры MuseScore.

### Диапазон камыля

- **Регистр 0** (малая октава / нижний регистр): A3, Bb3, C4, D4
  - Мёртвая зона: E4, F4, G4 не играются
- **Регистр 1** (первая октава): A4, Bb4, C5, D5
- **Регистр 2** (вторая октава): E5, F5, G5, A5
- **Регистр 3** (вторая-третья октава / передувы): Bb5, C6, D6

**Примечание:** Некоторые ноты имеют альтернативные аппликатуры в разных регистрах с разными пальцевыми позициями.

### Система обозначений

Камыль использует внутреннюю систему обозначений: буква ноты + номер регистра (например, A0, Bb1, C2, D3), где цифра обозначает регистр инструмента, а не октаву в научной нотации.
- 3 кружка (отверстия): ● = закрыто, ○ = открыто
- Цифра регистра под кружками

**Примечание о диапазоне воспроизводимых нот:** Ноты, которых нет в диапазоне камыля, будут отображаться красным цветом в MuseScore и не получат табулатуру.

## Установка

Подробная инструкция по установке инструмента Kamyl и плагина табулатуры находится в файле [INSTALL.md](INSTALL.md).ние

1. Откройте партитуру в MuseScore
2. Выделите такты, к которым хотите добавить табулатуру (или не выделяйте ничего для обработки всей партитуры)
3. Перейдите в меню: Plugins → Kamyl Tablature
4. Табулатура будет автоматически добавлена под нотами

### Отмена изменений

Если результат вас не устраивает, используйте: Edit → Undo (Ctrl+Z)

## Особенности

- Табулатура размещается под нотным станом
- Табулатура добавляется для всех нот, включая связанные (tie)
- Некоторые ноты (A5, D6) имеют альтернативные аппликатуры в разных регистрах - плагин показывает табулатуру для каждого появления ноты
- Поддержка грейс-нот (grace notes) будет добавлена в будущих версиях

## Известные ограничения

- Камыль -  инструмент и играет только определенные ноты. Хроматические ноты (диезы/бемоли) и ноты в мёртвой зоне будут отображаться красным цветом в MuseScore
- Ноты в мёртвой зоне (E4, F4, G4 / MIDI 64-68) физически не играются на камыле
- Грейс-ноты пока не поддерживаются
- Требуется MuseScore версии 3.0 или выше

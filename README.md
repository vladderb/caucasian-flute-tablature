# Kamyl Tablature Plugin для MuseScore

Плагин для автоматического добавления аппликатуры (табулатуры) для камыля (черкесской флейты) в партитуры MuseScore.

Поддерживаются **MuseScore 3.x** и **MuseScore 4.x**.

## Камыль (Kamyl)

Традиционная черкесская флейта с 3 отверстиями.

**Диапазон:**
- **Регистр 0**: A3, Bb3, C4, D4
- **Мёртвая зона**: E4, F4, G4 не играются
- **Регистр 1**: A4, Bb4, C5, D5
- **Регистр 2**: E5, F5, G5, A5
- **Регистр 3**: Bb5, C6, D6

**Хроматические ноты (с полузакрытыми отверстиями ◑):**
- **Регистр 0**: B3, C#4/Db4
- **Регистр 1**: B4, C#5/Db5
- **Регистр 2**: F#5, G#5/Ab5
- **Регистр 3**: B5, C#6/Db6
- **Регистр 4**: D#6

## Установка

Подробная инструкция по установке находится в файле [INSTALL.md](INSTALL.md)

**Краткая инструкция:**
1. Добавьте инструмент через Score Orders List 2 (Edit → Preferences → Score)
2. Скопируйте файлы плагинов в папку Plugins (используйте `kamyl_add_tablature_ms3.qml` для MuseScore 3 или `kamyl_add_tablature_ms4.qml` для MuseScore 4)
3. Перезапустите MuseScore

## Использование

### Добавление табулатуры

1. Откройте партитуру в MuseScore
2. Убедитесь, что партия использует инструмент Kamyl
3. Выделите такты, к которым хотите добавить табулатуру (или не выделяйте ничего для обработки всей партитуры)
4. Перейдите в меню: **Plugins** → **Kamyl Tablature**
5. Табулатура будет автоматически добавлена под нотами

### Удаление табулатуры

1. Откройте партитуру с табулатурой
2. Выделите такты, из которых хотите удалить табулатуру (или не выделяйте ничего для удаления из всей партитуры)
3. Перейдите в меню: **Plugins** → **Remove Kamyl Tablature**
4. Вся табулатура будет удалена

### Отмена изменений

Используйте: **Edit** → **Undo** (Ctrl+Z)

## Система обозначений табулатуры

- **●** (U+25CF) - закрытое отверстие
- **○** (U+25CB) - открытое отверстие
- **◑** (U+25D1) - полузакрытое отверстие (для хроматизмов)
- **?** - нота вне диапазона или в мёртвой зоне
- Цифра под кружками - номер регистра

## Особенности

- Табулатура размещается под нотным станом
- При повторном запуске плагина старые табы автоматически удаляются и заменяются новыми
- Плагин работает только с инструментом Kamyl
- Ноты вне диапазона отображаются с символом "?" вместо кружков
- Отдельный плагин для удаления табулатуры
- Поддержка MuseScore 3 и MuseScore 4

## Известные ограничения

- Плагин работает только с инструментом Kamyl (kamyl или wind.flutes.kamyl)
- Требуется MuseScore версии 3.0 или выше
- Альтернативные аппликатуры не поддерживаются

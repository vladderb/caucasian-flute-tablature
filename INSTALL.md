# Инструкция по установке плагина Kamyl Tablature

## Требования

- MuseScore версии 3.0 или выше (рекомендуется 3.6+ или 4.x)
- Операционная система: Windows, macOS или Linux

## Часть 1: Установка инструмента Kamyl

### Для MuseScore 4

**Важно:** MuseScore 4 имеет ограниченную поддержку кастомных инструментов. Официальная функция добавления через instruments.xml была удалена, но есть рабочий способ через Score Orders List 2.

#### Способ: Через Score Orders List 2 (сработал лично у меня)

1. Откройте MuseScore 4
2. Перейдите в **Edit** → **Preferences** (или **MuseScore** → **Preferences** на Mac)
3. Найдите раздел **Score**
4. Найдите опцию **"Score Orders List 2"**
5. Нажмите кнопку для добавления файла
6. Укажите путь к файлу `instruments.xml` ( если у вас уже используется там файл с кастомными инструментами то надо добавить камыль из этого файла в ваш уже используемый файл)
7. Нажмите **OK** и перезапустите MuseScore

После этого инструмент Kamyl появится в списке инструментов при создании партитуры (раздел Winds → Flutes).


### Для MuseScore 3

### Шаг 1: Определение папки инструментов MuseScore

- **Windows**: `C:\Program Files\MuseScore 3\instruments\`
- **macOS**: `/Applications/MuseScore 3.app/Contents/Resources/instruments/`
- **Linux**: `/usr/share/mscore-3.0/instruments/`

### Шаг 2: Копирование файла инструмента

1. Скопируйте файл `kamyl-instrument.xml` в папку instruments
2. Может потребоваться права администратора

### Шаг 3: Перезапуск MuseScore

Перезапустите MuseScore, чтобы новый инструмент появился в списке.

### Шаг 4: Добавление инструмента в партитуру

1. Создайте новую партитуру или откройте существующую
2. Нажмите `I` или перейдите в **Edit** → **Instruments** (Правка → Инструменты)
3. В поиске введите "Kamyl"
4. Добавьте инструмент в партитуру

## Часть 2: Установка плагинов

В комплект входят два плагина:
- `kamyl_tablature.qml` - добавление табулатуры
- `kamyl_tablature_remove.qml` - удаление табулатуры

### Шаг 1: Скачивание плагинов

Скачайте оба файла `.qml` из репозитория.

### Шаг 2: Определение папки плагинов

Папка для плагинов зависит от вашей операционной системы и версии MuseScore:

#### Windows

- **MuseScore 4**: `C:\Users\[ИмяПользователя]\Documents\MuseScore4\Plugins\`
- **MuseScore 3**: `C:\Users\[ИмяПользователя]\Documents\MuseScore3\Plugins\`

#### macOS

### Для MuseScore 3

#### Способ 1: Через файл instruments.xml (Рекомендуется)

1. Найдите папку данных MuseScore 3:
   - **Windows**: `C:\Users\[ИмяПользователя]\AppData\Local\MuseScore\MuseScore3\`
   - **macOS**: `~/Library/Application Support/MuseScore/MuseScore3/`
   - **Linux**: `~/.local/share/MuseScore/MuseScore3/`

2. Скопируйте файл `kamyl-instrument.xml` в эту папку и переименуйте его в `instruments.xml`
   - Если файл `instruments.xml` уже существует, откройте его и добавьте содержимое из `kamyl-instrument.xml` внутрь тега `<museScore>`

3. Перезапустите MuseScore 3

4. Создайте новую партитуру и найдите "Kamyl" в списке инструментов

#### Способ 2: Через настройки (Альтернативный)

1. Откройте MuseScore 3
2. Перейдите в **Edit** → **Preferences**
3. Откройте вкладку **Score**
4. В поле **Instrument list** укажите путь к файлу `kamyl-instrument.xml`
5. Нажмите **OK** и перезапустите MuseScore

### Добавление инструмента в партитуру

1. Создайте новую партитуру или откройте существующую
2. Нажмите `I` или перейдите в **Edit** → **Instruments**
3. В поиске введите "Kamyl"
4. Добавьте инструмент в партитуруина
7. Нажмите **OK**

#### Для MuseScore 3:

1. Запустите MuseScore 3
2. Откройте меню: **Plugins** → **Plugin Manager**
3. Найдите в списке **Kamyl Tablature**
4. Поставьте галочку напротив плагина
5. Нажмите **OK**

### Шаг 5: Проверка установки

1. Откройте любую партитуру или создайте новую
2. В меню должен появиться пункт: **Plugins** → **Kamyl Tablature**
3. Если пункт появился - установка прошла успешно!

## Устранение проблем

### Плагин не появляется в меню

1. Убедитесь, что файл `kamyl_tablature.qml` находится в правильной папке
2. Проверьте, что файл имеет расширение `.qml`, а не `.qml.txt`
3. Перезапустите MuseScore
4. Попробуйте нажать кнопку "Reload plugins" в настройках

### Плагин выдает ошибку при запуске

1. Убедитесь, что у вас установлена версия MuseScore 3.0 или выше
2. Проверьте консоль MuseScore на наличие сообщений об ошибках (View → Console)
3. Убедитесь, что файл плагина не поврежден

### На Linux: проблемы с правами доступа

Если плагин не запускается на Linux, возможно нужно установить права на выполнение:

```bash
chmod +x ~/Documents/MuseScore4/Plugins/kamyl_tablature.qml
```

## Устранение проблем

### Инструмент Kamyl не появляется в списке

**Для MuseScore 4:**
1. Убедитесь, что файл `kamyl-instrument.xml` добавлен через Score Orders List 2
2. Перезапустите MuseScore полностью
3. Если не помогает, используйте альтернативный способ: выберите любой духовой инструмент (Flute) и переименуйте его в партитуре

**Для MuseScore 3:**
1. Проверьте, что файл находится в правильной папке
2. Убедитесь, что XML-файл корректный (откройте в текстовом редакторе)
3. Перезапустите MuseScore

### Плагин не появляется в меню

1. Убедитесь, что файлы `kamyl_tablature.qml` и `kamyl_tablature_remove.qml` находятся в правильной папке
2. Проверьте, что файлы имеют расширение `.qml`, а не `.qml.txt`
3. Перезапустите MuseScore
4. Попробуйте нажать кнопку "Reload plugins" в настройках (если доступна)

### Плагин выдает ошибку при запуске

1. Убедитесь, что у вас установлена версия MuseScore 3.0 или выше
2. Проверьте консоль MuseScore на наличие сообщений об ошибках (View → Console)
3. Убедитесь, что файл плагина не поврежден

### На Linux: проблемы с правами доступа

Если плагин не запускается на Linux, возможно нужно установить права на выполнение:

```bash
chmod +x ~/Documents/MuseScore4/Plugins/kamyl_tablature.qml
```

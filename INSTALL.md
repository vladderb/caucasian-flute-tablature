# Инструкция по установке плагина Kamyl Tablature

## Требования

- MuseScore версии 3.0 или выше (MuseScore 3.x, 4.x)
- Операционная система: Windows, macOS или Linux

## Часть 1: Установка инструмента Kamyl

### Способ: Через Score Orders List 2 (работает для всех версий MuseScore)

1. Откройте MuseScore
2. Перейдите в **Edit** → **Preferences** (или **MuseScore** → **Preferences** на Mac)
3. Найдите раздел **Score**
4. Найдите опцию **"Score Orders List 2"** (или похожую опцию для добавления инструментов)
5. Нажмите кнопку для добавления файла
6. Укажите путь к файлу `instruments.xml` из этого проекта
   - Если у вас уже используется файл с кастомными инструментами, откройте его в текстовом редакторе и добавьте содержимое из нашего `instruments.xml` внутрь тега `<InstrumentGroup>`
7. Нажмите **OK** и перезапустите MuseScore

После этого инструмент **Kamyl** появится в списке инструментов при создании партитуры (раздел Woodwinds → Flutes).

### Добавление инструмента в партитуру

1. Создайте новую партитуру или откройте существующую
2. Нажмите `I` или перейдите в **Edit** → **Instruments**
3. В поиске введите "Kamyl"
4. Добавьте инструмент в партитуру
5. Нажмите **OK**

## Часть 2: Установка плагинов

В комплект входят четыре плагина:

**Для MuseScore 3:**
- `kamyl_add_tablature_ms3.qml` - добавление табулатуры
- `kamyl_remove_ms3.qml` - удаление табулатуры

**Для MuseScore 4:**
- `kamyl_add_tablature_ms4.qml` - добавление табулатуры
- `kamyl_remove_ms4.qml` - удаление табулатуры

### Шаг 1: Определение папки плагинов

Папка для плагинов зависит от вашей операционной системы и версии MuseScore:

#### Windows

- **MuseScore 4**: `C:\Users\[ИмяПользователя]\Documents\MuseScore4\Plugins\`
- **MuseScore 3**: `C:\Users\[ИмяПользователя]\Documents\MuseScore3\Plugins\`

#### macOS

- **MuseScore 4**: `~/Documents/MuseScore4/Plugins/`
- **MuseScore 3**: `~/Documents/MuseScore3/Plugins/`

#### Linux

- **MuseScore 4**: `~/Documents/MuseScore4/Plugins/`
- **MuseScore 3**: `~/.local/share/MuseScore/MuseScore3/plugins/`

### Шаг 2: Копирование файлов плагинов

1. Скопируйте нужные файлы `.qml` в папку плагинов в зависимости от вашей версии MuseScore
2. Убедитесь, что файлы имеют расширение `.qml`, а не `.qml.txt`

### Шаг 3: Перезапуск MuseScore

Перезапустите MuseScore, чтобы плагины появились в меню.

### Шаг 4: Проверка установки

1. Откройте любую партитуру или создайте новую
2. В меню должны появиться пункты:
   - **Plugins** → **Kamyl Tablature**
   - **Plugins** → **Remove Kamyl Tablature**
3. Если пункты появились - установка прошла успешно!

## Использование плагинов

### Добавление табулатуры

1. Откройте партитуру с инструментом Kamyl
2. Выделите такты, к которым хотите добавить табулатуру (или не выделяйте ничего для обработки всей партитуры)
3. Перейдите в меню: **Plugins** → **Kamyl Tablature**
4. Табулатура будет автоматически добавлена под нотами

### Удаление табулатуры

1. Откройте партитуру с табулатурой
2. Выделите такты, из которых хотите удалить табулатуру (или не выделяйте ничего для удаления из всей партитуры)
3. Перейдите в меню: **Plugins** → **Remove Kamyl Tablature**
4. Вся табулатура будет удалена

## Устранение проблем

### Инструмент не появляется в списке

1. Убедитесь, что файл `instruments.xml` добавлен через Score Orders List 2
2. Перезапустите MuseScore полностью
3. Проверьте, что XML-файл корректный (откройте в текстовом редакторе)

### Плагины не появляются в меню

1. Убедитесь, что файлы `.qml` находятся в правильной папке плагинов
2. Проверьте, что файлы имеют расширение `.qml`, а не `.qml.txt`
3. Убедитесь, что используете правильные файлы для вашей версии MuseScore
4. Перезапустите MuseScore
5. Проверьте консоль MuseScore на наличие ошибок (View → Console)

### Плагин выдает ошибку при запуске

1. Убедитесь, что у вас установлена версия MuseScore 3.0 или выше
2. Проверьте консоль MuseScore на наличие сообщений об ошибках
3. Убедитесь, что файл плагина не поврежден

### На Linux: проблемы с правами доступа

Если плагин не запускается на Linux, установите права на выполнение:

```bash
chmod +x ~/Documents/MuseScore4/Plugins/*.qml
chmod +x ~/.local/share/MuseScore/MuseScore3/plugins/*.qml
```
